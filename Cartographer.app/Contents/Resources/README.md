# Cartographer App Resources

## Custom Icon

To add a custom icon:

1. Create an icon file (.icns format) using:
   - SF Symbols app (free from Apple)
   - Icon Composer (Xcode)
   - Online tools like https://cloudconvert.com/png-to-icns

2. Name it `AppIcon.icns`

3. Place it in this Resources directory

4. The app will automatically use it!
